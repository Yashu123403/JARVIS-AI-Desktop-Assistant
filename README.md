JARVIS – Desktop Assistant

JARVIS is a lightweight, privacy-focused desktop assistant designed to enable seamless voice-based interaction with a computer. It uses speech recognition and automation to perform everyday tasks through simple voice commands.

This project was developed as a **college mini project** and later improved with additional features for better performance and usability.


FEATURES

- Voice Recognition (Speech-to-Text)
- Text-to-Speech Responses
- Desktop Automation (Open/close applications)
- Web Search & Information Retrieval
- Time-based queries and basic reminders
- Fully local processing for privacy


TECH STACK

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Python
- speech_recognition
- pyttsx3
- pyautogui
- subprocess

EXAMPLE COMMANDS
- Open Google Chrome
- Play "particular" song
- Open MS Word
